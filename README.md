# QR Code File Encoder

Полная CLI утилита для генерации QR‑кодовых файлов с минимальными метаданными.

## Структура проекта

```
qurqur/
├── qr_encoder.py          # Главный исполняемый файл (CLI)
├── services/
│   ├── __init__.py        # Пакетные файлы
│   ├── text_processor.py  # Обработка текста и метки
│   ├── qr_generator.py    # Генерация QR-кодов
│   ├── file_manager.py    # Работа с файлами
│   └── qr_collector.py    # Сборка QR-кодов
├── utils/
│   └── __init__.py
└── requirements.txt       # Зависимости
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск программы

```bash
python qr_encoder.py
```

## Основные функции

### 1. Интеграция текста в QR-коды
- Автоматическое разделение текста на блоки
- Минимальные метки: `#QRSTART:#` и `#QREND#`
- Метаданные: `FILEPATH:path BLOCKID:id TIME:timestamp CHECKSUM:checksum`

### 2. Настройка QR-кодов
- **Версия** (1-40): Определяет размер в модулях (21×21 до 177×177)
- **Коррекция ошибок**: L (7%), M (15%), Q (25%), H (30%)
- **Стиль**: Square (стандартные квадраты) или Circle (круглые модули)

### 3. Дополнительные функции
- Метаданные над/под каждым QR-кодом
- Обработка длинного текста (обрезка или перенос)
- Форматы сохранения: PNG, JPEG, BMP, SVG, PDF
- Группировка: сетки QR-кодов или отдельные файлы

### 4. Режим сбора данных
- Считывание QR-кодов из папки
- Интерактивный ввод данных
- Автоматическое восстановление исходного файла

## Пример метаданных

Формат минимальных метаданных:
```
FILEPATH:/path/to/file.txt BLOCKID:3a4b5c6d TIME:1234567890 CHECKSUM:0012
```

- `#QRSTART:` — начало блока данных
- `FILEPATH:` — путь к исходному файлу
- `BLOCKID:` — уникальный идентификатор блока (8 символов)
- `TIME:` — метка времени в Unix-формате
- `CHECKSUM:` — контрольная сумма размера в формате 4 цифры
- `#QREND#` — конец блока данных

## Использование

1. Запустите программу: `python qr_encoder.py`
2. Укажите текстовый файл для обработки
3. Настройте параметры QR-кода (версия, коррекция ошибок, стиль)
4. Выберите размещение метаданных и формат выходных файлов
5. Для группировки: выберите размер сетки (3x2 по умолчанию)
6. Программа автоматически создаст QR-коды в папке `qr_output/`

## Сборка файлов обратно

Для восстановления исходного файла из QR-кодов:
1. Сканируйте QR-коды на устройстве
2. Передайте программам через консоль или папку с файлами
3. Метаданные автоматически соберутся и восстановят исходный текст
