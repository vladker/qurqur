# QR Code File Encoder v2.0

Полная CLI утилита для генерации QR‑кодовых файлов с минимальными метаданными и восстановления файлов из QR-кодов.

**Поддерживает любые файлы:** текстовые, бинарные, изображения, документы, архивы.

## Возможности

### Генерация QR-кодов
- Кодирование текстовых файлов (UTF-8)
- Кодирование бинарных файлов (Base64)
- Автоматическая архивация для сжатия данных
- Выбор метода сжатия: ZIP, GZIP, BZ2, LZMA или автовыбор
- Разбиение на блоки с метаданными
- Настройка версии QR, коррекции ошибок, стиля

### Восстановление из QR-кодов
- Автоматическое распознавание QR-кодов из изображений
- Извлечение метаданных (путь, номер блока, тип файла, сжатие)
- Автоматическая разархивация
- Сохранение с оригинальным именем файла
- Проверка целостности блоков

## Основные функции

### 1. Интеграция текста в QR-коды
- Автоматическое разделение текста на блоки
- Минимальные метки: `#QRSTART:#` и `#QREND#`
- Метаданные: `FILEPATH:path BLOCKID:id TIME:timestamp CHECKSUM:checksum`

### 2. Настройка QR-кодов
- **Версия** (1-40): Определяет размер в модулях (21×21 до 177×177)
- **Коррекция ошибок**: L (7%), M (15%), Q (25%), H (30%)
- **Стиль**: Square (стандартные квадраты) или Circle (круглые модули)

### 3. Дополнительные функции
- Метаданные над/под каждым QR-кодом
- Обработка длинного текста (обрезка или перенос)
- Форматы сохранения: PNG, JPEG, BMP, SVG, PDF
- Группировка: сетки QR-кодов или отдельные файлы

### 4. Режим сбора данных
- Считывание QR-кодов из папки
- Интерактивный ввод данных
- Автоматическое восстановление исходного файла

## Пример метаданных

**Внутри QR-кода** (компактный формат):
```
BN:1 TOT:25 M:B C:zip #QRS#...данные...#QRE#
```

- `BN:` — номер блока (1-999)
- `TOT:` — всего блоков
- `M:` — тип (T=текст, B=бинарный/Base64)
- `C:` — метод сжатия (none, zip, gzip, bz2, lzma)
- `#QRS#` — начало данных (QR Start)
- `#QRE#` — конец данных (QR End)

**Над QR-кодом** (текстом):
```
filename.ext | Блок 1/25 | 1234567890
```
- Имя исходного файла
- Номер блока и общее количество
- Unix timestamp

## Использование

1. Запустите программу: `python qr_encoder.py`
2. Укажите текстовый файл для обработки
3. Настройте параметры QR-кода (версия, коррекция ошибок, стиль)
4. Выберите размещение метаданных и формат выходных файлов
5. Для группировки: выберите размер сетки (3x2 по умолчанию)
6. Программа автоматически создаст QR-коды в папке `qr_output/`

## Восстановление файла из QR-кодов

1. Запустите программу: `python qr_decoder.py`
2. Укажите путь к папке с изображениями QR-кодов
3. Программа автоматически:
   - Просканирует все изображения в папке (PNG, JPG, JPEG, BMP, GIF, TIFF, WEBP)
   - Распознает QR-коды с помощью библиотеки pyzbar
   - Извлечёт метаданные и содержимое блоков
   - Проверит наличие всех блоков
   - Восстановит исходный текст
4. Выберите способ сохранения:
   - Сохранить в файл (предлагается путь по умолчанию)
   - Вывести в консоль
